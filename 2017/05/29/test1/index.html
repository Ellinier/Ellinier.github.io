Introduction

Related Work

Definition : slip s

$$ s = \frac{\omega r - v}{\omega r} $$

estimate wheel slip:

  • analyze wheel encoder or torque data: impractical

  • Visual Odometry (VO): computational expensive, impact drive speed; poor accuracy on featureless terrains and shadowed areas

  • stochastic modeling: rely on GPS positive data

  • machine learning methods

  • SVM

  • ANN

  • SOM algorithm

  • classification on proprioceptive sensor data

  • vibration-based classifiers

  • gyroscopes: gravel, pavement, dirt

  • motor current: sand

  • microphone: grass

slippage estimation framework

  • three slip classes

  • low slip: $ s \leq 0.3$

  • moderate slip: $ 0.3 < s \leq 0.6$

  • high slip: $ s > 0.6$

  • data preprocessed

  • filter: low-pass filters(低通滤波)

  • removing the unneeded data(野点剔除)

  • synchronizing the clocks(同步时钟)–synchronizing the clocks of the two logging computers

  • calculate features

  • angular velocity of each wheel and the linear velocity of the rover

  • performance analyze

  • accuracy, precision, recall, $F_1$ score, ROC curve and confusion matrix

  • testing time

  • storage requirements

  • test performance

  • cross-validation method
